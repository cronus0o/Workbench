**Project Concept**

linux terminal practice program

This program will help a person who is not know about shell command script.

Main Page (initial window)

**side tool bar**
- info (explain program)
- 
- test cli


Once I execute the program, system autometically detects users login history.
And, pop up login-screen
If the password is corect, pop up main window.

In main window, there are four button (help, cli, gui, history) in the left side tool bar.
Side tool bar is in the most left of the screen.
User can click each tool button.

Except the side tool bar space, all the other space is interactive-screen space.
When user clicks each side tool bar button, button's own interactive-screen pop up in interactive-screen.

1. first interactive screen about 'help' (side tool bar).
In this screen, there is information of linux, shell script, 'how to use this program' (I'll write this later)    

2. second interactive screen about 'cli' 
In this screem, user can write command with their keyborad.
And, program system will autometically examine typed code and print output.

3. third interactive screen about 'gui'
In this screen, user can see file hierarchy system that user modified with GUI.
I want to make this like window 11's file tracker

4. fourth interactive screen about 'history'
In this screen, there is the list of command that user inserted
User can examine what they did.
